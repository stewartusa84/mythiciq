# MythicIQ Roadmap

MythicIQ is nearing beta. The core replay system is working, and the current focus is polishing the desktop app, testing LFG, and building the trust systems that support clean runs, badges, and better groups for Mythic+ and raid.

---

## Current Progress

### ✅ Core Replay System

Local combat-log parsing and replay review are working for run review, with raid support growing from the same foundation.

### 🟣 Near Beta

We are currently finishing the desktop app flow, LFG testing, stat verification, and the first version of clean run credit.

### ⬜ Public Release

Full public release will follow once the desktop app, LFG, addon bridge, and trust systems are stable enough for wider use.

### ⬜ Mobile Companion

Mobile apps are planned after the full desktop release.

## Long-Term Goal

MythicIQ is being built as a local-first companion for reviewing Mythic+ runs and raid pulls, improving play, and finding groups with aligned expectations.

The goal is simple: better tools for group content without turning the game into another public shame board or ad-filled score site.
